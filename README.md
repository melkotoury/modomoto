**Modomoto**  _Task_ 



* **CORE**

A small ruby app for manipulating a dataset coming from a csv file

* **STEPS**

1. place your corrupted csv file in the project root directory
2. run the following command ```ruby main.rb```



* **TECHNOLOGIES USED** 

1. Ruby 2.51 
2. RVM
3. RubyMine
4. CSV Library
5. Rubocop
